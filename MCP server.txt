Enable MCP Server Running on port 9876

Allow trading via MCP

Require confirmation for trading operations

Setup Instructions.

Claude Code / Cursor — add mp json to your project root:

{
  "mcpServers": {
    "ctrader": {
      "type": "http",
      "url": "http://127.0.0.1:9876/mcp/"
    }
  }
}

For other MCP clients: connect to http://127.0.0.1:0876/mep/
